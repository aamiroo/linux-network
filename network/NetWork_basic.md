
شبکه یا نت ورک (NetWork) به معنای برقراری ارتباط با دیگران و تبادل اطلاعات مد نظر در قالب یک فضای متمرکز صورت میگیرد.

# 📌 انواع شبکه

۱. LAN: مخفف Local Area Network شبکه محلی
۲. WAN : مخفف Wide Area Network شبکه در مقیاس بسیار وسیع
۳. MAN : مخفف  Metropolitan Area Network  شبکه شهری

این سه مدل از شبکه متداول ها هستند و اصلی ترین تفاوت آنها گستره عملکردی آنها است.
___

# 🌐 شبکه ها چگونه ایجاد و کامپیوتر ها چگونه متصل میشوند؟

در مبحث شبکه هر دستگاه یا کامپیوتری که قرار است به شبکه متصل شود یا شده یک host یا node نام دارد.
اولین راه ارتباطی host با شبکه از طریق اتصالات فیزیکی یا همان کابل ها صورت میگیرید.
این کابل ها به دستگاهی به نام سوییچ switch متصل هستند.
### سوییچ چیست؟
سوییچ دستگاهی است که ارتباط بین دیوایس ها (همون host ها) با ip ها را در یک شبکه محلی (LAN) برقرار میکند که بر اساس  **MAC Address** میباشد.
پس دومین راه ارتباطی در شبکه با دستگاه سوییچ است.

### روتر (Router) چیست؟
سومین مسیر ارتباطی در شبکها از طریق روتر صورت میگیرد. روتر برای خارج شدن یک هاست و برقراری ارتباط با شبکه ای به غیر از شبکه محلی خود آن هاست صورت میگیرد. درواقع ارتباط بین شبکه ها نیز با روتر انجام میشود.

___
# چگونه در یک شبکه ارتباط برقرار میکنیم؟

برای فهمیدن این موضوع باید مفاهیم زیر را بلد باشیم:

۱. IP یا Internet Protocol
۲. Subnet mask
۳. CIDR
۴. NAT
۵. Port
۶. DNS
۷. Fire wall

### آی پی (IP) و CIDR , Subnet mask
شناسه دیوایس های متصل به شبکه که به صورت زیر نمایش داده میشود:

```
(Net ID + Host ID)
192.168.1.1/24    # برای مثال
```

نت آی دی در شبکه همیشه ثابت می باشد و هاست آی دی برای هر دیوایسی متفاوت و مختص خود آن دیوایس است.
با توجه به مثال بالا خواهیم داشت:

Net ID : 192.168.1
Host ID : 1
در آی پی  X/ را CIDR مینامیم و از ۰ تا ۳۲ داریم. که نمایانگر طول شبکه نت ورک میباشد
___
### انواع IP و مدل های IP گرفتن

ما دو نوع IP داریم
- **خصوصی (Private IP)**:
این مدل IP بصورت اختصاصی و فقط در شبکه محلی (LAN) مورد استفاده قرار میگرید.

- **عمومی (Public IP)**:
همان IP که در اینترنت قابل مشاهده و در دسترس است.

**چگونه IP دریافت میکنیم؟**
دیوایس ها به دو صورت خودکار (DHCP) و دستی (Manual Configuration) IP خود را دریافت میکنند.

- **حالت (DHCP)**:
در این حالت دستگاه پس از اولین اتصال به شبکه به صورت خودکار درخواست IP میکند. این درخواست از  DHCP server خواهد بود.

- **حالت دستی (Manual Configuration)**:
در این حالت مدیر سیستم یه صورت دستی IP وارد میکند.

**نسخه های IP**
- **IPv4**:
همان IP شناخته شده و معمولی است. از ۳۲ بیت تشکیل شده است.
مثال:

```
192.168.1.1
```

- **IPv6**:
به دلیل کمبود آدرس‌های IPv4 معرفی شده است و ظرفیت بسیار بیشتری دارد. از ۱۲۸ بیت تشکیل شده.
نمایش آن به صورت زیر است:

```
2001:db8:85a3::8a2e:370:7334
```
___
### دی ان اس (DNS)
دی ان اس در شبکه نوشته ها (دامنه) را به IP و یا بالعکس تبدیل میکنید.
برای مثال وقتی سرچ میکنیم google.com دی ان اس آن را به **74.125.224.72** تبدیل میکند. این همان گوگل است ام به شکل IP
___
### نت (NAT)
نت (NAT) یا نت ورک آدرس ترنسلیشن (Network Address Translation ) برای تبدیل IP خصوصی در شبکه محلی به IP عمومی یا IP gateway استفاده میشود تا با فضای بیرون از شبکه محلی نیز ارتباط صورت بگیرد اما خود IP نمایش داده نشود.
این کار توسط روتر انجام میشود.
___
### پورت و فایروال ( Port , Fire wall )

پورت در واقع دروازه ورود به شبکه و سرویس هاست که انواع مختلفی دارد و ما برای وارد شدن به شبکه ها باید از انها با IP که داریم عبور کنیم.
فایر وال نیز برای کنترل ترافیک ورودی و خروجی همین پورت ها استفاده میشود و درواقع اجازه دسترسی ایجاد میکند برای یک پورت شبکه که آیا میتوان وارد آن شد یا خیر. فایروال ها نیز نواع مختلفی دارند.
