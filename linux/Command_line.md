
برای استفاده از سیستم و دسترسی به کل کامپیوتر در فضای غیر گرافیکی (ترمینال یا CLI) از دستورات یا command ها استفاده میکنیم.

برای نمونه:
```bash
ls 
pwd
lsblk
uname
w
kill
nc
```

در ادامه به طور تخصصی و به شکل دسته بندی شده برای هر موضوع  به این دستورات خواهیم پرداخت:


# مسیر فایل ها

مسیر فایل محل قرارگیری یک فایل یا پوشه در سیستم‌عامل را مشخص می‌کند. با دستورات مربوط به مسیر می‌توان بین پوشه‌ها جابه‌جا شد یا مسیر فعلی را مشاهده کرد و و یا لیستی از آن داشت.

‍‍‍‍```
```bash
pwd                    # نمایش مسیر فعلی
```

```bash
ls                  # لیست فایل ها و پوشه
```

```bash
ls -l             # نمایش لیست با جزییات
```

```bash
ls -la              # نمایش همه حتی مخفی
```

```bash
cd                      # رفتن به مسیر
```

```bash
cd ..              # رفتن به پوشه قبلی
```

___
# فایل و دایرکتوری

فایل‌ها محل ذخیره اطلاعات در لینوکس هستند. خود فایل ها نیز درون دایرکتوری ها قرار دارند. با دستورات مربوط به فایل می‌توان فایل ایجاد کرد، حذف کرد، کپی گرفت، تغییر نام داد یا اطلاعات آن را مشاهده نمود.

```bash
tuoch               #ساختن فایل خالی
```

```bash
mkdir               # ساخت دایرکتوری
```

```bash
rm                 # حذف فایل
```

```bash
cp                 # کپی فایل یا دایرکتوری
```

```bash
mv                 # جابجایی با تغییر نام
```

# دسترسی به مجوزها (Permissions)

در لینوکس هر فایل و پوشه دارای سطح دسترسی مشخصی است که تعیین می‌کند چه کاربرانی می‌توانند آن را بخوانند، ویرایش کنند یا اجرا کنند.

```bash
ls -l           # نمایش مجوزهای فایل
```

```bash
chmod +x        # اضافه کردن مجوز اجرایی 
```

____

# پروسه ها (Processes)

پروسه به هر برنامه یا دستوری گفته می‌شود که در حال اجرا است. دستورات این بخش برای مشاهده، مدیریت و پایان دادن به پردازش‌های سیستم استفاده می‌شوند.

```bash
ps aux         # نمایش تمام پروسه ها
```

```bash 
top
htop          # نمایش زنده پروسه ها
btop
```

```bash
kill          # بستن یک پروسه
kill -9       # بسیتن اجباری یک پروسه
```

# دی مسیج ها

در لینوکس برای نمایش **پیام‌های کرنل (Kernel Ring Buffer)** استفاده می‌شود. این پیام‌ها شامل اطلاعات مربوط به بوت سیستم، شناسایی سخت‌افزار، درایورها، خطاها و هشدارها هستند.

```bash
dmesg        # نمایش تمام پیام های کرنل
```

```bash
dmesg -H     # نمایش خروجی به شکل خوانا تر 
```

```bash
dmesg | less  # مشاهده خروجی صفحه به صفحه
```

```bash
dmesg | grep usb    # نمایش پیام های فقط مربوط به یو اس بی
```

```bash
dmesg -T      # نمایش زمان پیام‌ها به‌ صورت تاریخ و ساعت قابل خواندن
```

# مدیریت حافظه و فضا

برای نمایش فضای مصرق شده در سیستم و به طور کلی مدیریت فضا و پارتیشن بنده هارد ها از این درستورات استفاده میکنیم.

```bash 
lsblk         # نمایش هاردها و پارتیشن بتدی
```

```bash
df -h         # نمایش فضای استفاده شده و باق مانده
```

```bash
du -sh        # نمایش حافظه یک پوشه
```

```bash
free -h       # نمایش سواپ
```
